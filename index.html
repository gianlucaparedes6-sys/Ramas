Diccionario Git Interactiva

Aprendiendo a trabajar con ramas

#

git init

Inicializa un nuevo repositorio local en la carpeta actual.

#

git clone

Descarga una copia exacta de un repositorio remoto.

#

git add

Prepara los cambios (staging) para ser incluidos en el próximo commit.

#

git commit

Guarda los cambios preparados en el repositorio local.
